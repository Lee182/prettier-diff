# prettier-diff
A vue js project, that transforms code with prettier, then can a see a diff with diff2html.

## install & start
```fish
git clone https://github.com/Lee182/prettier-diff.git & cd prettier-diff
pnpm install
pnpm run build
pnpm run start
```
